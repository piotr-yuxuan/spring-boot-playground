# `com.github.piotryuxuan.springbootplayground`

This is a simple project to demonstrate usage of:
  - Java 21 (LTS)
  - Spring Boot
  - React (single-page application)
  - PostgreSQL

# Getting Started

``` zsh
mvn spring-boot:run
mvn spring-boot:start
mvn spring-boot:stop
```
